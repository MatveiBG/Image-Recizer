# Image-Recizer
The goal of this project was to be able to take a bulk of images and recize them all to the same size without loosing any part of the image.

Current features:
- Choose recize size
- Choose a whole folder to recize
- GUI
- Remoce Background

Requirements:
- python: >3.7, <3.12 (for rembg)
- rembg 
- numpy
- pillow
- tkinter
